## Data Acquisition Strategy

### Step 1: Get Episode List with IDs
```bash
# Use YouTube Data API or yt-dlp to get full episode list
yt-dlp --flat-playlist --print "%(id)s %(title)s" \
  "https://www.youtube.com/@FoundersPodcast/videos" > founders_episodes.txt
```

### Step 2: Match Priority List to Video IDs
- Cross-reference episode numbers with YouTube titles
- Create priority download list

### Step 3: Download Priority 100
```bash
# Download only priority episodes as audio
yt-dlp -x --audio-format mp3 --audio-quality 0 \
  -o "%(title)s.%(ext)s" \
  --download-archive downloaded.txt \
  -a priority_video_ids.txt
```

### Step 4: Transcribe with Parakeet
```bash
# Run Parakeet on downloaded audio
python transcribe_batch.py --input ./audio --output ./transcripts
```
